✨ ::before와 ::after 완벽 가이드

1️⃣ 기본 개념: ::before와 ::after란?

💡 ::before와 ::after는 HTML을 추가하지 않고 CSS만으로 요소를 만드는 가상 요소입니다.
이것이 실제 HTML 내용입니다 
/* CSS */
.basic-box::before {
    content: '👉 ';  /* 앞에 추가 */
}

.basic-box::after {
    content: ' ✅';  /* 뒤에 추가 */
}

/* HTML - 실제로는 이것만 작성 */
<div class="basic-box">
    이것이 실제 HTML 내용입니다
</div>

/* 브라우저가 이렇게 렌더링: */
👉 이것이 실제 HTML 내용입니다 ✅

2️⃣ 구조 이해하기

📦 요소의 구조: ::before → 실제 내용 → ::after
이것이 실제 HTML 내용
(div.visual-demo) 
.visual-demo::before {
    content: '::before (앞에 추가됨)';
    position: absolute;
    top: -30px;
    /* ... */
}

.visual-demo::after {
    content: '::after (뒤에 추가됨)';
    position: absolute;
    bottom: -30px;
    /* ... */
}

    top: 0;

}
❓ 왜 HTML로 div 하나 더 안 만들고 ::before를 쓸까?
✅ HTML이 깔끔해짐 (구조와 장식 분리)
✅ 재사용 가능 (CSS만 바꾸면 됨)
✅ JavaScript로 조작할 필요 없는 순수 장식 요소

4️⃣ 실전: 말풍선 화살표

💡 ::before로 삼각형을 만들어 화살표처럼 보이게!
툴팁 메시지 
.tooltip::before {
    content: '';
    position: absolute;
    bottom: -10px;
    left: 50%;

    /* CSS로 삼각형 만들기 (border 트릭) */
    width: 0;
    height: 0;
    border-left: 10px solid transparent;
    border-right: 10px solid transparent;
    border-top: 10px solid #3B82F6;
}

}

📌 핵심 정리

필수 규칙

규칙 설명 예시
content 필수 content가 없으면 표시 안 됨 content: ''; 또는 content: '텍스트';
::는 2개 구버전 :before도 작동하지만 ::before 권장 ::before ✅ :before ⚠️
inline 요소 주의 <span>에는 absolute/fixed 사용 어려움 display: inline-block; 추가

자주 쓰는 패턴

  • 🎨 오버레이: content: ''; position: absolute;
  • ✏️ 텍스트/아이콘: content: '✓';
  • 📐 도형: content: ''; border 트릭;
  • ✨ 장식 효과: transition과 hover 조합

team-card에서 왜 사용했나?

목적: 배경 이미지 위에 검은색 반투명 레이어 추가
이유: HTML에 div 추가하지 않고 CSS만으로 해결
장점: 코드가 깔끔하고 재사용 가능

💡 ::before 없이 vs 있을 때

방법 코드 장단점
HTML div 추가 
<div class="card">
  <div class="overlay"></div>
  <div class="content">...</div>
</div>
 ✅ 초보자에게 이해하기 쉬움
❌ HTML이 복잡해짐
❌ JavaScript로 조작 시 선택자 필요
::before 사용 
<div class="card">
  <div class="content">...</div>
</div>

.card::before {
  content: '';
  /* overlay CSS */
}
 ✅ HTML 깔끔
✅ CSS로 완전 제어
✅ 재사용 용이
⚠️ content 필수
