방법 코드 특징 추천
1. Container container mx-auto px-4 ✅ 자동으로 반응형 max-width
✅ 가장 간단
✅ Tailwind 표준 		⭐⭐⭐
2. Max-width max-w-7xl mx-auto px-4 ✅ 최대 너비 직접 제어
✅ 유연함
⚠️ max-w 값 선택 필요 		⭐⭐
3. Margin mx-4 md:mx-8 lg:mx-16 ✅ 완전한 제어
⚠️ 여백 크기 직접 관리
⚠️ 코드 복잡 		⭐

💡 추천

일반적인 경우: container mx-auto px-4 lg:px-8

최대 너비 제어: max-w-7xl mx-auto px-4

📱 모바일 vs 데스크톱

모바일: px-4 (좌우 16px 여백)
데스크톱: lg:px-8 (큰 화면에서 32px 여백)

🔍 시각적 비교

화면 크기별 동작:

📱 모바일 (0~767px)

← px-4 (16px) 여백 → 콘텐츠 전체 폭 ← px-4 →

🖥️ 데스크톱 (1280px 이상)

좌측 여백
max-w-7xl (1280px) 콘텐츠 영역
우측 여백
